## 一、系统概述

### 1.1 什么是三级分销

三级分销是一种基于社交关系的裂变式销售模式，消费者可以成为推广者，通过分享商品链接给他人购买，获得相应佣金奖励。最多支持三级分销关系，即：

```
A → B → C → D
│   │   │
│   │   └─ D 购买，C 获得一级佣金
│   └───── C 购买，B 获得一级佣金，A 获得二级佣金
└───────── B 购买，A 获得一级佣金
```

### 1.2 核心优势

- 📈 **裂变增长**：用户即推广者，实现病毒式传播
- 💰 **多方共赢**：平台、推广者、消费者三方受益
- 🎯 **精准营销**：基于信任关系的推荐，转化率更高
- 📊 **数据驱动**：完整的追踪统计，优化运营策略

## 二、分销层级设计

### 2.1 层级关系定义

| 层级 | 关系描述 | 佣金比例 | 说明 |
|------|----------|----------|------|
| 一级分销 | 直接邀请关系 | 10% - 15% | A 直接邀请 B，B 消费 A 获得一级佣金 |
| 二级分销 | 间接邀请关系 | 5% - 8% | A 邀请 B，B 邀请 C，C 消费 A 获得二级佣金 |
| 三级分销 | 深度邀请关系 | 2% - 4% | A→B→C→D，D 消费 A 获得三级佣金 |

### 2.3 层级绑定规则

1. **绑定触发**：用户通过推广链接/二维码首次访问并注册时，自动绑定上下级关系
2. **绑定保护期**：注册后 15 天内未完成首单，关系可被其他推广者覆盖
3. **永久绑定**：完成首单后，分销关系永久锁定
4. **跨级限制**：最多追溯三级，超过三级不再分配佣金

## 三、佣金比例和分配机制

### 3.1 佣金配置表

| 商品类型 | 一级佣金 | 二级佣金 | 三级佣金 | 合计成本 |
|----------|----------|----------|----------|----------|
| 实物商品 | 12% | 6% | 3% | 21% |
| 虚拟商品 | 15% | 8% | 4% | 27% |
| 服务类 | 10% | 5% | 2% | 17% |
| 特价商品 | 5% | 2% | 1% | 8% |

### 3.2 佣金计算公式

```
一级佣金 = 订单实付金额 × 一级佣金比例
二级佣金 = 订单实付金额 × 二级佣金比例
三级佣金 = 订单实付金额 × 三级佣金比例

实付金额 = 商品总价 - 优惠券 - 平台折扣 + 运费
```

### 3.3 佣金分配示例

**场景**：A → B → C → D 的四级关系链，D 购买了 1000 元的商品（实物类）

| 受益人 | 关系层级 | 佣金比例 | 佣金金额 |
|--------|----------|----------|----------|
| C | 一级 | 12% | 120 元 |
| B | 二级 | 6% | 60 元 |
| A | 三级 | 3% | 30 元 |
| 平台 | - | - | 790 元（扣除其他成本前）|

### 3.4 佣金结算规则

| 状态 | 说明 | 结算时间 |
|------|------|----------|
| 待结算 | 订单已支付，等待收货确认 | - |
| 可结算 | 订单确认收货，过售后期 | 确认收货后 7 天 |
| 已结算 | 佣金已打入账户余额 | - |
| 已提现 | 佣金已提现到银行卡/微信 | - |
| 已冻结 | 订单退款/退货，佣金冻结 | - |
| 已失效 | 退款完成，佣金收回 | - |

### 3.5 提现规则

- **最低提现**：50 元起提
- **提现手续费**：1%（平台承担或用户承担可配置）
- **到账时间**：
  - 微信零钱：实时到账
  - 银行卡：1-3 个工作日
- **提现限制**：
  - 单日限额：5000 元
  - 单月限额：50000 元
  - 实名认证后方可提现

## 五、推广追踪和统计

### 5.1 追踪方式

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| 推广链接 | 带参数的专属链接 | 社交媒体、聊天工具 |
| 二维码 | 专属推广二维码 | 线下场景、海报 |
| 邀请码 | 6 位数字/字母邀请码 | 手动输入场景 |
| 小程序码 | 微信小程序码 | 微信生态内 |

### 5.2 链接参数设计

```
https://example.com/product/123?inviter=A001&channel=wechat&campaign=spring2026

参数说明：
- inviter: 邀请人 ID（必填）
- channel: 推广渠道（wechat/weibo/douyin/direct）
- campaign: 活动标识（可选）
- source: 流量来源（可选）
```

### 5.3 数据统计维度

#### 5.3.1 推广数据

| 指标 | 说明 |
|------|------|
| 曝光量 | 推广链接/二维码被查看次数 |
| 点击量 | 推广链接被点击次数 |
| 点击率 | 点击量 / 曝光量 |
| 注册数 | 通过推广注册的用户数 |
| 转化率 | 注册数 / 点击量 |
| 下单数 | 推广用户下单数量 |
| 成交金额 | 推广订单总金额 |

#### 5.3.2 佣金数据

| 指标 | 说明 |
|------|------|
| 预计佣金 | 订单完成前的预估佣金 |
| 可提现佣金 | 已过售后期的佣金 |
| 已提现佣金 | 已提出去的佣金 |
| 佣金收益率 | 佣金 / 推广成交金额 |

#### 5.3.3 团队数据

| 指标 | 说明 |
|------|------|
| 直推人数 | 一级下级数量 |
| 团队总人数 | 所有下级总数量 |
| 活跃代理数 | 近 30 天有产出的下级 |
| 团队总业绩 | 团队累计成交金额 |

## 六、防作弊机制

### 6.1 常见作弊手段

| 作弊类型 | 手段描述 | 危害程度 |
|----------|----------|----------|
| 自买自推 | 自己注册多个账号，自己购买 | ⭐⭐⭐⭐⭐ |
| 虚假交易 | 下单后退款，套取佣金 | ⭐⭐⭐⭐⭐ |
| 刷单刷量 | 机器刷点击、刷注册 | ⭐⭐⭐⭐ |
| 恶意抢客 | 覆盖他人已绑定用户 | ⭐⭐⭐ |
| 羊毛党 | 专门薅优惠券和新人福利 | ⭐⭐⭐ |
| 黑产账号 | 使用虚拟号码、接码平台 | ⭐⭐⭐⭐⭐ |

### 6.2 技术防范措施

#### 6.2.1 设备指纹

```
采集维度：
- 设备 IMEI/IDFA
- 设备型号、系统版本
- 屏幕分辨率
- 浏览器指纹
- IP 地址
- GPS 位置（可选）

风险判定：
- 同一设备注册多个账号 → 高风险
- 同一 IP 大量注册 → 中风险
- 设备频繁切换账号 → 高风险
```

#### 6.2.2 行为分析

| 异常行为 | 判定规则 | 处理方式 |
|----------|----------|----------|
| 短时大量注册 | 同一 IP 1 小时注册>10 个 | 限制注册，人工审核 |
| 异常下单 | 新账号立即大额下单 | 延迟结算，人工审核 |
| 高频退款 | 退款率>50% | 冻结佣金，限制提现 |
| 集中时间下单 | 多账号同一分钟下单 | 标记为可疑订单 |
| 异常浏览 | 无浏览直接下单 | 降低佣金比例 |

#### 6.2.3 关系链检测

```
检测规则：
1. 环路检测：A→B→A 的循环关系禁止
2. 自我推广：下单人与推广人为同一人禁止
3. 设备关联：同一设备多账号关联下单预警
4. 地址关联：同一收货地址多账号下单预警
5. 支付关联：同一支付方式多账号下单预警
```

### 6.3 业务防范措施

#### 6.3.1 佣金冻结策略

| 风险等级 | 冻结比例 | 冻结时长 | 解冻条件 |
|----------|----------|----------|----------|
| 低风险 | 0% | - | - |
| 中风险 | 30% | 15 天 | 无退款自动解冻 |
| 高风险 | 100% | 30 天 | 人工审核解冻 |
| 黑名单 | 100% | 永久 | 不予解冻 |

#### 6.3.2 提现限制

```
提现前校验：
1. 实名认证状态 - 必须完成实名认证
2. 近 30 天退款率 - 超过 30% 限制提现
3. 账户活跃度 - 注册未满 7 天限制大额提现
4. 异常行为检测 - 触发风控规则需人工审核
5. 提现频率 - 每日最多 3 次提现申请
```

#### 6.3.3 黑名单机制

```
列入黑名单条件：
1. 确认自买自推行为
2. 恶意刷单累计 3 次以上
3. 使用黑产工具/接码平台
4. 被举报核实 3 次以上
5. 关联账号存在严重作弊

黑名单后果：
- 永久冻结所有未提现佣金
- 禁止参与任何分销活动
- 禁止注册新账号（设备/IP 封禁）
- 情节严重者移交司法机关
```
