三、租赁服务条款

3.1 租赁流程

  1. 选择设备并提交租赁订单
  2. 完成实名认证和信用评估
  3. 支付租金和押金（如需）
  4. 签署电子租赁合同
  5. 接收设备并开始使用
  6. 租期结束后归还设备

3.2 用户义务

在租赁期间，您应当：

  • 妥善保管和使用租赁设备
  • 不得擅自转租、抵押、出售设备
  • 不得对设备进行拆解、改装
  • 按时支付租金
  • 租期结束后按时归还设备

3.3 损坏赔偿

如设备在租赁期间发生损坏或丢失：

  • 正常磨损：无需赔偿
  • 人为损坏：按维修费用赔偿
  • 设备丢失：按设备采购价赔偿
  • 从押金中扣除相应费用，不足部分需补足

四、Token 服务条款

4.1 Token 购买

  • Token 一经售出，除特殊情况外不支持退款
  • Token 有有效期，请在有效期内使用
  • Token 不可转让、不可兑换现金

4.2 使用规范

您不得利用 Token 服务：

  • 生成违法、有害、侵权内容
  • 进行自动化批量请求攻击
