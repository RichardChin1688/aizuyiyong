## 页面间导航

所有页面均包含统一的顶部导航栏，支持快速跳转到任意业务页面：

```
首页 ←→ 设备租赁 ←→ 算力套餐 ←→ AI 开发 ←→ 学习论坛 ←→ 企业服务 ←→ 分销合作 ←→ 用户中心
```

### 导航组件特性

- **统一样式**：所有页面使用相同的导航栏设计
- **高亮当前**：自动高亮当前所在页面
- **响应式设计**：支持桌面端和移动端
- **快速链接**：每个页面底部包含快速导航卡片

## 技术栈

### 当前实现
- **HTML5** - 静态页面结构
- **Tailwind CSS** (CDN) - 样式框架
- **响应式设计** - 移动端适配

### 推荐生产环境
参考 `../architecture.md` 完整技术架构：
- **Next.js 14+** - React 框架（App Router）
- **TypeScript** - 类型安全
- **shadcn/ui** - UI 组件库
- **Tailwind CSS** - 样式框架
