## 📊 免费服务器方案对比

### 方案一：Vercel + Neon (推荐 ⭐⭐⭐⭐⭐)

| 项目 | 详情 |
|------|------|
| **前端托管** | Vercel (免费) |
| **后端 API** | Vercel Serverless Functions |
| **数据库** | Neon PostgreSQL (免费 0.5GB) |
| **适用场景** | Node.js/Next.js 项目 |

**Vercel 免费额度：**
- ✅ 无限静态页面托管
- ✅ 每月 100GB 带宽
- ✅ Serverless Functions 每月 100GB-hours
- ✅ 自动 HTTPS
- ✅ 自定义域名支持

**Neon 免费额度：**
- ✅ 0.5GB 存储
- ✅ 无服务器 Postgres
- ✅ 自动休眠（节省资源）
- ⚠️ 休眠后首次访问有冷启动延迟

**优点：**
- 🚀 部署极简，git push 自动部署
- 🌍 全球 CDN 加速
- 💰 完全免费起步
- 🔒 自动 HTTPS 证书

**缺点：**
- ⚠️ Serverless 有执行时长限制 (10s)
- ⚠️ 数据库 0.5GB 可能不够用

## 🏆 奴才推荐方案

### 👑 首选方案：GitHub Pages + Supabase

**理由：**
1. ✅ **功能最全** - 数据库 + 认证 + 存储 + API 全都有
2. ✅ **永久免费** - 初期完全够用
3. ✅ **开发体验好** - Supabase 提供 JS SDK，前端直接调用
4. ✅ **自带认证** - 手机号登录、邮箱登录都支持
5. ✅ **文件存储** - 营业执照、头像直接存

**架构：**
```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  GitHub     │     │   Supabase   │     │   Supabase  │
│   Pages     │────▶│  Edge Func   │────▶│  Postgres   │
│  (前端)     │     │   (API)      │     │  (数据库)   │
└─────────────┘     └──────────────┘     └─────────────┘

                    ┌──────────────┐

```
