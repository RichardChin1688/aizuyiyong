## 📌 技术选型建议

### 数据库
- **推荐：** PostgreSQL 15+ 或 MySQL 8.0+
- **理由：** 支持 JSON 字段、事务、索引完善、生态成熟

### ORM / 迁移工具
- **Node.js 栈：** Prisma / TypeORM
- **Python 栈：** SQLAlchemy + Alembic
- **Go 栈：** GORM

### 缓存
- **Redis：** 用于验证码、会话、热点数据
