❓ 常见问题

如何开始设备租赁？

+

1. 浏览设备租赁页面，选择您需要的设备

2. 选择租期（日租/周租/月租/年租）

3. 完成实名认证和信用评估

4. 支付租金和押金（信用良好可免押）

5. 签署电子合同，等待发货

Token 套餐有哪些？价格如何？

+

我们提供多种 Token 套餐：

• 体验包：10,000 Token - ¥15（7 天有效）

• 灵活包：100,000 Token - ¥120（30 天有效）

• 标准包：1,000,000 Token - ¥1000（90 天有效）

• 企业包：10,000,000 Token - ¥8000（180 天有效）

还有月付订阅套餐，详情请查看 Token 工厂页面。

如何申请发票？

+

1. 登录用户中心

2. 进入"订单管理"

3. 找到需要开票的订单

4. 点击"申请发票"

5. 填写发票抬头和税号

6. 选择电子发票或纸质发票

电子发票将在 1-3 个工作日内发送到您的邮箱。

设备损坏了怎么办？

+

1. 立即联系客服报备情况

2. 不要自行维修设备

3. 我们将安排专业检测

4. 根据检测结果确定赔偿金额

5. 从押金中扣除，不足部分需补足

正常磨损无需赔偿，人为损坏按维修费用或折旧价赔偿。

如何调用 Token API？

+

1. 在用户中心创建 API Key

2. 参考 API 文档集成到您的应用

3. 使用 OpenAI 兼容接口格式

4. 在请求头中携带 API Key

示例代码：

curl https://api.ai-zuyiyong.com/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"gpt-4o","messages":[{"role":"user","content":"Hello"}]}'

支持哪些支付方式？

+

我们支持以下支付方式：

• 支付宝（个人/企业）

• 微信支付

• 企业对公转账

• 信用卡/储蓄卡

大额订单（超过 10000 元）支持分期付款，详情请联系客服。
